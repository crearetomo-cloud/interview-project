# DM Request Template

## Purpose
Initial outreach via direct message to request an interview.

## Template

**Subject/Opening:** [Keep it brief and personal]

Hi [Name],

I've been following your work on [specific topic/project] and I'm really impressed by [specific reason]. 

I'm working on [project name/description] and would love to feature your insights. Would you be open to a [duration] interview about [specific topic]?

I'd be happy to share more details about the project and how your story would fit in.

Best,
[Your Name]

## Tips

- Personalize each message - mention something specific about their work
- Keep it concise (3-4 sentences max)
- Make the ask clear but not pushy
- Include a clear call to action
- Follow up once if no response after 1 week

## Follow-up Template

Hi [Name],

Just wanted to follow up on my message from [date]. I know you're busy, but I'd love to chat about [topic] if you have a moment.

No pressure at all - just wanted to make sure the message didn't get lost in your inbox.

Thanks!
[Your Name]
