# Introduction Request Template

## Purpose
Request an introduction from a mutual connection.

## Template

Hi [Mutual Connection's Name],

Hope you're doing well! I'm reaching out because I'm working on [project description] and would love to interview [Target Person's Name] about [topic].

I noticed you're connected with [Target Person] on [platform]. Would you be comfortable making an introduction? I'd be happy to draft a message you could forward, or if you prefer, just a brief intro would be great.

I think [Target Person] would be a perfect fit because [reason].

Let me know if this works for you!

Thanks,
[Your Name]

## Tips

- Only ask for introductions from people you have a genuine relationship with
- Make it easy for them - offer to draft the message
- Explain why the introduction makes sense
- Be respectful if they decline
- Always thank them, even if they can't help
