# Interview Checklist

## Pre-Interview Checklist

### Research
- [ ] Review interviewee's background
- [ ] Read their recent work/publications
- [ ] Check their social media presence
- [ ] Identify key topics to discuss
- [ ] Note any recent news/achievements

### Preparation
- [ ] Draft interview script
- [ ] Prepare 10-15 core questions
- [ ] Prepare 5-10 follow-up questions
- [ ] Review consent form
- [ ] Prepare recording equipment
- [ ] Test recording setup
- [ ] Charge devices/backup battery
- [ ] Prepare backup recording method

### Logistics
- [ ] Send calendar invite
- [ ] Confirm time zone
- [ ] Send meeting link/details
- [ ] Send consent form (if needed in advance)
- [ ] Block time for post-interview work
- [ ] Set reminder for thank you message

## Interview Day Checklist

### Before Interview
- [ ] Review script and notes
- [ ] Test recording equipment
- [ ] Close unnecessary applications
- [ ] Ensure good internet connection
- [ ] Have water/notes ready
- [ ] Join call 2-3 minutes early

### During Interview
- [ ] Confirm recording permission
- [ ] Review consent form
- [ ] Start recording
- [ ] Take notes on key points
- [ ] Stay within time box
- [ ] Ask follow-up questions
- [ ] Thank interviewee

### After Interview
- [ ] Stop recording
- [ ] Save recording immediately
- [ ] Back up recording
- [ ] Send thank you message (within 24 hours)
- [ ] Note any immediate insights

## Post-Interview Checklist

### Content Creation
- [ ] Transcribe recording (if needed)
- [ ] Review transcript/notes
- [ ] Identify key quotes
- [ ] Draft article structure
- [ ] Write first draft
- [ ] Edit for clarity and flow
- [ ] Check facts and quotes
- [ ] Write headline options
- [ ] Final proofread

### Review Process
- [ ] Send draft to interviewee
- [ ] Set review deadline
- [ ] Incorporate feedback
- [ ] Confirm final approval
- [ ] Make final edits

### Publication
- [ ] Prepare publishing copy
- [ ] Select/request images
- [ ] Final proofread
- [ ] Publish article
- [ ] Share link with interviewee
- [ ] Request share (optional)
- [ ] Archive all materials

## Quality Checks

- [ ] All quotes are accurate
- [ ] Facts are verified
- [ ] No typos or grammatical errors
- [ ] Article flows well
- [ ] Headline is compelling
- [ ] Images are appropriate and credited
- [ ] Links work correctly
- [ ] Interviewee has approved content
