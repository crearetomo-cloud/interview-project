# Interview Workflow

## Overview
This document outlines the complete workflow from initial outreach to final publication.

## Stages

### 1. Pre-Interview

- [ ] Identify potential interviewee
- [ ] Research their background and work
- [ ] Draft outreach message (see `docs/outreach/`)
- [ ] Send initial request
- [ ] Schedule interview (if accepted)
- [ ] Send calendar invite with details
- [ ] Prepare interview script (use template)
- [ ] Review consent form
- [ ] Test recording equipment/software

### 2. Interview

- [ ] Confirm recording permission
- [ ] Review consent form with interviewee
- [ ] Start recording (with permission)
- [ ] Follow interview script (flexible)
- [ ] Take notes on key points
- [ ] Ask follow-up questions
- [ ] Stay within time box
- [ ] Thank interviewee

### 3. Post-Interview

- [ ] Send thank you message (within 24 hours)
- [ ] Transcribe recording (if applicable)
- [ ] Review transcript/notes
- [ ] Draft article using template
- [ ] Identify key quotes and insights
- [ ] Write headline options
- [ ] Edit and refine draft
- [ ] Send draft to interviewee for review
- [ ] Incorporate feedback
- [ ] Finalize article

### 4. Publication

- [ ] Prepare publishing copy
- [ ] Select/request images
- [ ] Final proofread
- [ ] Publish article
- [ ] Share with interviewee
- [ ] Request share (optional)
- [ ] Archive materials

## Timeline

- **Outreach to Response**: 1-2 weeks
- **Scheduling**: 1-2 weeks in advance
- **Interview to Draft**: 1 week
- **Review Period**: 3-5 business days
- **Finalization to Publication**: 2-3 days
- **Total**: ~4-6 weeks from outreach to publication
